# GLOSSARY OF SYSTEM TERMS

## 1. Purpose
Define all important terms used across the system to ensure clarity and consistency.

## 2. Core Terms
**Workflow**  
A full cycle from INIT to COMPLETE.

**Orchestrator**  
The central controller coordinating all agents.

**Agent**  
A modular component with a single responsibility.

**Safety Invariant**  
A rule that must always be true for the system to operate safely.

**Circuit Breaker**  
A mechanism that halts the system when safety is violated.

**Regime**  
Market condition classification (e.g., bullish, bearish).

**Signal**  
A recommendation generated by the analysis agent.

**Backtest Metrics**  
Historical performance indicators.

**Risk Veto**  
A decision from the risk agent blocking execution.

**Execution**  
Placing a paper trade.

**Audit Record**  
A complete log of the workflow.

## 4. Safety Terms
**Blocking Condition**  
A condition that prevents workflow continuation.

**Validation**  
A check ensuring data or output meets requirements.

**Failure Mode**  
A predictable way the system can fail safely.
